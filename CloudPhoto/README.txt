Процесс запуска: 
1) chmod 777 {ваш путь}/cloud_photo_linux/CloudPhoto

Пример, chmod 777 /usr/bin/cloud_photo_linux/CloudPhoto

2) Прописать алиас
cd ~
nano .bashrc
alias cloudphoto='{ваш путь}/cloud_photo_linux/CloudPhoto' 
Пример, alias cloudphoto='/usr/bin/cloud_photo_linux/CloudPhoto'

Сохранить изменения.
